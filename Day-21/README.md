Day 21 - 40 Days of JavaScript - Build a Quiz Game using DOM

🎯 Goal of This Lesson

1.✅ Complete the design of a Quiz App.
2.✅ Develop the Quiz App with the following features:
(i). Show Random Questions for the Quiz.
(ii). Start a timer for the X(say 15) seconds for user to answer a question.
(iii). After 15 seconds a Next button should appear to navigate to the next question.
(iv). If user is not selecting an answer within the 15 seconds, the right answer should be selected by default.
(v). When user selects a wrong answer, the right answer should be shown.
(vi). A result should be shown at the end of all the questions.
(vii). The result should also store the highest scorer and show that.
(viii). There shold be a way to restart the game at the end.

1.✅ Non Functional Requirements:
(i). Good Look and Feel.
(ii). Optimal usages of the timer.
(iii). Clean code.
